# MOS Service (Multimedia Object Storage System)

Spring Boot 微服務，用於處理 MOS 系統的檔案下載連結取得和檔案上傳功能。

## 功能特性

- 取得檔案下載連結 (API 1-4)
- 上傳檔案 (API 1-5)
- Docker 容器化部署
- RESTful API
- 輸入參數驗證

## 技術棧

- Java 11
- Spring Boot 2.7.18
- Maven
- Docker & Docker Compose
- Lombok
